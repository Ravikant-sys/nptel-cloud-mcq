# NPTEL Cloud Computing MCQ Platform

[![Live Site](https://img.shields.io/badge/Live%20Site-nptel--cloud--mcq.onrender.com-blue?style=for-the-badge&logo=render)](https://nptel-cloud-mcq.onrender.com/)
A high-performance, interactive MCQ testing platform designed specifically for the **NPTEL Cloud Computing** course assignments. This project provides a premium user interface to practice and master cloud concepts through interactive weekly quizzes and comprehensive grand tests.

## 🚀 Key Features

- **Interactive Weekly Tests**: Dedicated modules for each assignment week.
- **Grand Test Mode**: Combine and shuffle all questions from the entire semester for final exam preparation.
- **Premium Aesthetics**: Features a modern **Glassmorphism UI** with a dynamic, animated background (Space/Cloud themes).
- **Light & Dark Modes**: Responsive theme switching with specialized background animations (hills/clouds for day, stars for night).
- **Mobile Optimized**: Fully responsive layout for studying on any device.

## 📊 Engagement Success

The initial live deployment of the Cloud Computing platform has seen significant community engagement:
- **250+ Visitors**: Reached over 250 unique visitor hits during the initial live phase.
- **Visitor Tracking**: Integrated with CounterAPI for real-time engagement monitoring via a secure namespace.

## 🛠️ Technical Overview

- **Frontend**: Vanilla HTML5, CSS3, and JavaScript (ES6+).
- **Styling**: Modern CSS Grid and Flexbox with heavy use of CSS variables for dynamic theming.
- **Data Driven**: Centralized assignment data management for easy updates.
- **Hosting**: Render (Static Site Distribution).
- **API**: [CounterAPI](https://counterapi.dev/) for anonymous visitor tracking.

## 📦 Usage

1. Open `index.html` in any modern web browser.
2. Select the **Cloud Computing** course (now expanded to support Multi-Course selection).
3. Choose a specific week or jump into the **Grand Test**.
4. Track your score and review detailed solutions upon submission.
